## Ключевые пользовательские потоки

<div align="center">

| **Поток**              | **Актор**    | **Описание**                                    |
| :--------------------- | :----------- | :---------------------------------------------- |
| Создание сессии        | Модератор    | Создание комнаты, настройка колоды и параметров |
| Присоединение к сессии | Участник     | Вход в комнату по ссылке или коду приглашения   |
| Голосование            | Участник     | Выбор карты для оценки задачи в текущем раунде  |
| Открытие результатов   | Модератор    | Показ голосов участников для обсуждения         |
| Сброс голосования      | Модератор    | Перезапуск раунда для повторной оценки          |
| Завершение сессии      | Модератор    | Сохранение результатов и экспорт оценок         |
| Просмотр истории       | Пользователь | Доступ к завершённым сессиям и прошлым оценкам  |

</div>

## Маршруты приложения

<div align="center">

| **Путь**        | **Страница**     | **Описание**                                         |
| :-------------- | :--------------- | :--------------------------------------------------- |
| `/`             | `OnboardingPage` | Приветственная страница: описание проекта, CTA       |
| `/create-room`  | `CreateRoomPage` | Создание комнаты: название, имя, выбор колоды        |
| `/room/:roomId` | `RoomPage`       | Игровая комната: голосование, результаты, управление |
| `/login`        | `LoginPage`      | Вход в аккаунт                                       |
| `/register`     | `RegisterPage`   | Регистрация нового пользователя                      |
| `/dashboard`    | `DashboardPage`  | Список комнат пользователя (активных и завершённых)  |
| `/profile`      | `ProfilePage`    | Настройки профиля пользователя                       |
| `/about`        | `AboutPage`      | Информация о проекте                                 |

</div>

### Основной сценарий (без авторизации)

```
/ → Onboarding
    ↓ «Начать» или «Создать комнату»
/create-room → CreateRoomPage (форма: название, имя, колода)
    ↓ «Начать игру»
/room/:roomId → RoomPage (голосование, результаты)
```

### Сценарий с авторизацией

```
/ → Onboarding
    ↓ «Войти» / «Зарегистрироваться»
/login или /register
    ↓ Успешный вход
/dashboard → список комнат
    ↓ «Создать комнату»
/create-room → RoomPage
```

## Условные обозначения

Схема использует стандартные символы блок-схем с цветовой кодировкой для быстрого визуального восприятия.

### Фигуры

<div align="center">

| **Фигура**                  | **Значение**                                                            | **Примеры**                                               |
| :-------------------------- | :---------------------------------------------------------------------- | :-------------------------------------------------------- |
| ○ Овал                      | **Терминатор** — начало или конец потока                                | `Начало`                                                  |
| ▭ Прямоугольник             | **Процесс / Экран** — страница, экран или действие пользователя         | `Главная страница`, `Вход`, `Регистрация`, `Комната`      |
| ◇ Ромб                      | **Решение / Условие** — точка ветвления с бинарным выбором (Да/Нет)     | `Успешный вход?`, `Подтвердить выход?`, `Способ входа?`   |
| ▱ Параллелограмм            | **Ввод/Вывод** — внешнее действие: отправка, получение данных           | `Отправить ссылку`, `Неверные данные`                     |
| ⬡ Шестиугольник             | **Данные / Результат** — представление данных или сохранённый результат | `История комнат`, `Список комнат`, `Результат`, `История` |
| ▭ Скруглённый прямоугольник | **Терминатор данных** — начальные данные или конечный статус            | `Имя гостя`                                               |

</div>

### Цвета

<div align="center">

| **Цвет**         | **Значение**                                         | **Примеры**                                             |
| :--------------- | :--------------------------------------------------- | :------------------------------------------------------ |
| 🟢 Зелёный       | **Начало** — стартовая точка потока                  | `Начало`                                                |
| 🟡 Жёлтый        | **Экран / Процесс** — страница или основное действие | `Главная страница`, `Вход`, `Комната`, `Дашборд`        |
| 🔵 Голубой       | **Условие / Решение** — точка ветвления логики       | `Успешный вход?`, `Подтвердить выход?`, `Способ входа?` |
| 🔴 Красный       | **Ошибка** — некорректные данные или отказ           | `Неверные данные`                                       |
| Светло-бирюзовый | **Данные / Результат** — вывод или сохранение данных | `История комнат`, `Результат`, `Список комнат`          |

</div>

### Стрелки и подписи

<div align="center">

| **Подпись**                                       | **Значение**                               |
| :------------------------------------------------ | :----------------------------------------- |
| `Да` / `Нет`                                      | Результат проверки условия в ромбе         |
| `Успешно`                                         | Успешное завершение операции               |
| `Повторить`                                       | Возврат к предыдущему шагу для исправления |
| `Войти` / `Регистрация` / `Гость`                 | Путь авторизации пользователя              |
| `Создать` / `Открыть`                             | Действие с комнатой                        |
| `Покинуть` / `Завершить`                          | Выход из комнаты или завершение сессии     |
| `Отправить ссылку` → `Открыть ссылку приглашения` | Приглашение участника в комнату            |

</div>
